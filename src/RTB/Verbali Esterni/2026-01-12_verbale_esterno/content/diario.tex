\section{Diario della riunione}
\subsection{Spiegazione di base tecnologie}
L'azienda ha fornito una spiegazione di base sulle tecnologie che utilizzeremo durante il progetto. 
\subsubsection{Docker}
L'azienda ha spiegato l'architettura principale dietro a \vrs{Docker}, in particolare:
\begin{itemize}
    \item Nozioni di base sulla containerizzazione
    \item Creazione ed utilizzo di Dockerfile
    \item Utilizzo di file .env
    \item Creazione ed utilizzo di file DockerCompose
\end{itemize}
\subsubsection{FastAPI}
L'azienda ha spiegato l'architettura principale dietro a \vrs{FastAPI}, in particolare:
\begin{itemize}
    \item Struttura generale
    \item Principali vantaggi
    \item Creazione di un endpoint di base
\end{itemize}
\subsubsection{LangChain}
L'azienda ha spiegato l'architettura principale dietro a \vrs{LangChain}, in particolare:
\begin{itemize}
    \item Struttura di un progetto di base
    \item Utilizzo di alcune funzioni di base
    \item Utilizzo della piattaforma \vrs{LangSmith}
\end{itemize}
\subsubsection{React}
L'azienda ha spiegato l'architettura principale dietro a \vrs{React}, in particolare:
\begin{itemize}
    \item Struttura di una funzione di base
    \item Utilizzo del \vrs{DOM} virtuale
    \item Discussione di ulteriori librerie e alternative
\end{itemize}
